## 🎮 What is SuGa BLAZE?

SuGa BLAZE is a fast-paced **3D endless runner** where every second alive is a victory. Sprint through an ever-accelerating world, switch lanes, dodge everything that moves, grab every coin you can — and don't stop until the game stops you.

Built entirely from scratch in **Unity 6** with **C#**. This is **Project 4** of my #100DaysOfCode journey. There's more coming. 👀

---

## ✨ Features

| | Feature | Description |
|---|---|---|
| 🧍 | **Animated 3D Character** | Smooth run, jump & crouch animations |
| 🛣️ | **3-Lane System** | Fluid left/right lane switching |
| 🧱 | **Obstacle Variety** | Walls, barriers & lane blockers |
| 🪙 | **Coin Collection** | Coins at multiple heights — timing is key |
| ⚡ | **Progressive Speed** | Gets faster. And faster. And faster. |
| 🔊 | **Audio** | Background music + satisfying SFX |
| 📱 | **Full UI** | Main Menu → Pause → Game Over |
| 🏆 | **Best Score Persistence** | Your high score lives on between sessions |

## 🕹️ Controls

```
  ┌─────────────────────────────────────────┐
  │   A / ◄     ──   Switch Lane Left       │
  │   D / ►     ──   Switch Lane Right      │
  │   SPACE     ──   Jump                   │
  │   S / ▼     ──   Crouch                 │
  │   ESC       ──   Pause                  │
  └─────────────────────────────────────────┘
```

> 💡 **Tip:** The game gets faster over time. Don't get comfortable.
